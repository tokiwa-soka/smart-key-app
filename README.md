# スマート貸出管理アプリ (GitHubアップロード用)

作成したアプリのデータ一式です。このフォルダ内のファイルをGitHubにアップロードすることで、誰でもスマホからQRコードを使って利用できるようになります。

## 🚀 GitHub Pages への公開手順

1. GitHubにログインし、新しいリポジトリ（例: `smart-key-app`）を作成します。
2. 作成したリポジトリに、このフォルダ (`SmartKeyApp`) の中身（`index.html`, `admin.html`, `style.css`, `app.js` など）をすべてドラッグ＆ドロップでアップロードし、Commit（保存）します。
3. リポジトリの **[Settings]** > **[Pages]** を開きます。
4. **Source** を `Deploy from a branch` にし、**Branch** を `main` (または `master`) にして `Save` を押します。
5. 数分待つと、「Your site is live at `https://あなたのユーザー名.github.io/smart-key-app/`」というURLが発行されます。

## 📱 アプリの使い方

1. 発行されたURLの末尾に `admin.html` を付けたURL（例: `https://.../admin.html`）にパソコンからアクセスします。
2. 管理者ダッシュボードが開くので、**「QRコード生成用URL一覧」**に表示されている各鍵のURLをコピーします。
3. 無料のQRコード作成サイトでQRコード画像を作成し、印刷して鍵や保管場所に貼り付けます。
4. 職員が自分のスマホでそのQRコードを読み込むと、直接その鍵の貸出画面（または返却画面）が開きます！

---

## ⚠️ 【超重要】データベースの連携について（全職員で同期する場合）

現在このコードは、**「操作したスマホ本体（LocalStorage）」**にのみ貸出状態を保存する「単体動作モード」になっています。
つまり、Aさんが自分のスマホで貸出処理をしても、Bさんのスマホでは「貸出可」のまま見えてしまいます。

**全員のスマホでリアルタイムに状態を共有するには、データベースが必要です。**
職員の皆様が使い慣れている **「Google スプレッドシート」** をデータベースとして使うのが最も簡単で無料です。

### 📊 Google スプレッドシート連携の手順
1. Googleドライブで新しいスプレッドシートを作成し、「貸出状態」「履歴」などのシートを作ります。
2. メニューの「拡張機能」＞「Apps Script」を開き、簡単なプログラム（データを読み書きするAPI）を記述して「ウェブアプリとしてデプロイ（公開）」します。
3. デプロイして発行されたURLを、本アプリの `config.js` 内にある `GAS_URL = "ここにURLを貼る"` に設定して再度GitHubにアップロードします。
4. `app.js` の `fetchState()` や `saveState()` 関数を、LocalStorage ではなく `fetch(GAS_URL)` でスプレッドシートと通信するように少しだけ書き換えます。

※もし「スプレッドシート連携（本番環境用）のコードも書いてほしい！」という場合は、いつでもAIにお声がけください！具体的なApps Scriptのコードと、`app.js`の修正版を即座に作成いたします。
